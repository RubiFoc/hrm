# Bugfix

1. Reproduce the issue with a deterministic command.
2. Narrow root cause to one subsystem before editing.
3. Apply minimal fix, then add regression test.
4. Verify both affected and adjacent behavior.
5. Return root cause, fix summary, and evidence.
