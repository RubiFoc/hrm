# LLM Start Here

## Objective
Get enough context fast, implement safely, and leave the project easier to maintain.

## Startup Sequence
1. Read `docs/llm/context-map.md`.
2. Load only task-relevant docs from `docs/`.
3. Read `docs/engineering/best-practices.md`.
4. Fill `.ai/team/contracts/task-input.yaml` for the task.
5. Execute workflow from `.ai/team/workflow.md`.
6. Update docs using `docs/llm/docs-update-checklist.md`.

## Required Delivery Artifacts
- Implementation summary
- Test evidence
- Decision updates
- Documentation updates
- Diagram updates (if architecture or flow changed)
