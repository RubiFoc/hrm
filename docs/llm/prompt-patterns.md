# Prompt Patterns

## 1. Implementation Task
"Implement <feature> with constraints <...>. Use docs/llm/context-map.md to load context. Return changed files, test evidence, and docs updated."

## 2. Bugfix Task
"Reproduce and fix <bug>. Provide root cause, minimal fix, regression test, and updated runbook/testing docs if needed."

## 3. Review Task
"Review this diff for correctness, regressions, and missing tests. Report findings by severity with file references."
