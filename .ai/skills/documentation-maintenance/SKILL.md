# Documentation Maintenance

1. Identify impacted doc areas from `docs/README.md`.
2. Apply minimal updates in the same task as code/process changes.
3. Keep facts in source documents; avoid duplicated text.
4. Update decision log when architecture-impacting changes occur.
5. Run checklist from `docs/llm/docs-update-checklist.md`.
